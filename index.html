Grocery List Program

  • Adding items: type "add" followed by the item you want to add.
  • Removing items: type "remove" followed by the item.
  • Listing items: type the word "list".
  • Exiting: either type exit or close out of the prompt box.

Examples

  • add pineapple
  • add kiwi
  • remove pineapple
  • list
  • exit
